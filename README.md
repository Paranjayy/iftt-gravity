# 🌌 Gravity Hub: The God Build v4.6

Gravity is a production-grade, self-healing, autonomous home intelligence system designed for local execution and remote mission control. It bridges low-level hardware control (Mac, WiZ, MirAie) with high-fidelity visual reporting, environmental awareness, and sensory feedback.

*Note: Status cards are generated live via Puppeteer.*

## 🚀 Live Intelligence Features

- **🏏 IPL Match Center 2.0**: Tabbed UI (Live, Highlights, Wickets, Timeline). Syncs lighting (Red: Wicket, Gold: 6, Blue: 4).
- **📊 Momentum Meter**: Live [====|----] tracking based on CRR vs RRR.
- **🏆 Result Fallback**: Automatic end-of-innings summary with Top Performers and Next Match countdowns.
- **🔊 Audio Sensory Engine**: Audible cues for silent updates (Tink), new alerts (Glass), wickets (Basso), and boundaries (Ping).
- **🛡️ Unified Anti-Spam**: Edits existing messages for continuous updates (GitHub, Markets, ISS) to prevent chat flooding.
- **🐙 GitHub Pulse**: Purple pulse on activity, silent-toggle support, and CI/CD status monitoring.
- **📈 Market Vibe**: Real-time tracking of BTC/ETH & Tech Stocks. Alerts on 5% swings with "Moon Alert" logic.
- **🛰️ ISS Overhead**: White blink and alert when the ISS is above the Ahmedabad/Hub sector.
- **🔮 Prediction Oracle**: Tracks Polymarket/Kalshi sentiment shifts (Alerts on >15% probability swings).
- **🧠 Wikipedia Rabbit Hole**: Daily random knowledge deep-dives at 10 PM.
- **🌑 Moon Phase Mood**: Automatically dims lights based on the current lunar cycle (0-4 AM).
- **🌅 Golden Hour**: Automatic color matching for sunset sky vibes based on local sunset times.
- **🛡️ Focus Shield**: Red alert warning when opening blacklisted apps (Discord/Twitter/YouTube) on Mac.
- **📦 Delivery Watch**: Automated checking for delivery status (Simulated / Test signals supported).

## 🛠️ Command Reference

| Command | Action |
| :--- | :--- |
| `/ping` | Heartbeat + Platform Indicator (Local vs GitHub) |
| `/live` | Open the **Match Center** (Tabs: Live, Highlights, Wickets, Timeline) |
| `/control` | Open the **God Mode Dashboard** (AC/Lights/Stats/Pulse Toggles) |
| `/card` | Generate high-fidelity Visual Status Report |
| `/odds` | Check top prediction market odds (**Polymarket & Kalshi**) |
| `/follow` | Follow a prediction market keyword for alerts (e.g. `/follow gpt-5`) |
| `/unfollow`| Stop following a prediction market keyword |
| `/track` | Add stock/crypto to Market Pulse (Tip: `.NS` for India) |
| `/untrack`| Remove stock/crypto from Market Pulse |
| `/deliver` | Simulate a delivery arrival (Orange/Angry-Red alerts) |
| `/trending`| Scrape top GitHub repositories live by language |
| `/vibe` | Trigger environmental scenes (Cinema, Focus, Dinner, etc.) |
| `/timer` | Start a live countdown timer for any hardware device |
| `/once` | Set a temporary, self-destructing routine |

## 🌌 Deployment & Setup
Gravity is optimized for **Mac OS Sonoma** and requires **WiZ / MirAie** credentials in `.env`.

```bash
git clone https://github.com/Paranjayy/iftt-gravity.git
npm install
./gravity-launch.sh
```
