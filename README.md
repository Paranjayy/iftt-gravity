## 🚀 Live Intelligence Features

- **🏏 IPL Cricket Mode**: Syncs lighting with matches (Red: Wicket, Gold: 6, Blue: 4). Auto-scores with commentary.
- **🐙 GitHub Pulse**: Green pulse on commits, **3x Red Flash** on CI/CD workflow failure.
- **📈 Market Vibe**: Real-time tracking of BTC/ETH & Tech Stocks. Alerts on 3% swings.
- **🛰️ ISS Overhead**: Blink white when the International Space Station is above Ahmedabad.
- **🌅 Golden Hour**: Automatic color matching for sunset sky vibes.
- **🛡️ Focus Shield**: Red light warning when opening blacklisted apps (Discord/Twitter) on Mac.
- **🚀 Trending Scraper**: Use `/trending <lang>` to scrape top GitHub repos live.
- **⚛️ OSS Pulse**: Native integration with `@if2opensource_bot` for GitHub trending and major OSS releases.

## 🛠️ Command Reference

| Command | Action |
| :--- | :--- |
| `/ping` | Heartbeat + Platform Indicator (Local vs GitHub) |
| `/card` | Generate high-fidelity Visual Status Report |
| `/odds` | Check top prediction market odds (**Polymarket & Kalshi**) |
| `/follow` | Follow a prediction market keyword for alerts (e.g. `/follow gpt-5`) |
| `/unfollow`| Stop following a prediction market keyword |
| `/track` | Add stock/crypto to Market Pulse (Tip: `.NS` for India) |
| `/untrack`| Remove stock/crypto from Market Pulse |
| `/deliver` | Simulate a delivery arrival (for testing Orange/Angry-Red alerts) |
| `/once` | Set a temporary, self-destructing routine |
| `/timer` | Start a live countdown timer for any device |
| `/vibe` | Trigger environmental scenes (Cinema, Focus, Dinner, etc.) |
