## 🚀 Live Intelligence Features

- **🏏 IPL Match Center 2.0**: Tabbed UI for Live Score, Boundaries, Wickets, and Timeline. Syncs lighting (Red: Wicket, Gold: 6, Blue: 4).
- **🚀 Momentum Meter**: Real-time momentum tracking [====|----] based on CRR vs RRR.
- **🛡️ Unified Anti-Spam**: Edits existing messages for continuous updates (GitHub, Markets, ISS) to prevent chat flooding.
- **🔊 Sensory Engine**: Audible cues for silent updates (Tink), new alerts (Glass), wickets (Basso), and boundaries (Ping).
- **🐙 GitHub Pulse**: Purple pulse on activity, **3x Red Flash** on CI/CD workflow failure.
- **📈 Market Vibe**: Real-time tracking of BTC/ETH & Tech Stocks. Alerts on 5% swings.
- **🛰️ ISS Overhead**: Blink white when the International Space Station is above your Hub.
- **🧠 Wikipedia Rabbit Hole**: Daily random knowledge deep-dives at 10 PM.
- **🌑 Moon Phase Mood**: Automatically dims lights based on the current lunar cycle phase (0-4 AM).
- **🌅 Golden Hour**: Automatic color matching for sunset sky vibes.
- **🛡️ Focus Shield**: Red light warning when opening blacklisted apps (Discord/Twitter) on Mac.

## 🛠️ Command Reference

| Command | Action |
| :--- | :--- |
| `/ping` | Heartbeat + Platform Indicator (Local vs GitHub) |
| `/live` | Open the **Match Center** (Tabs: Live, Highlights, Wickets, Timeline) |
| `/card` | Generate high-fidelity Visual Status Report |
| `/odds` | Check top prediction market odds (**Polymarket & Kalshi**) |
| `/follow` | Follow a prediction market keyword for alerts (e.g. `/follow gpt-5`) |
| `/track` | Add stock/crypto to Market Pulse (Tip: `.NS` for India) |
| `/deliver` | Simulate a delivery arrival (Orange/Angry-Red alerts) |
| `/vibe` | Trigger environmental scenes (Cinema, Focus, Dinner, etc.) |

## 🌌 Deployment & Setup
Gravity is optimized for **Mac OS Sonoma** and requires **WiZ / MirAie** credentials in `.env`.

```bash
git clone https://github.com/Paranjayy/iftt-gravity.git
npm install
./gravity-launch.sh
```
